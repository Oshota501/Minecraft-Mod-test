# ビルド

いちいちMODフォルダに移動するのは手間なのでよかったらbuild.shを使ってください。一応下記の方法で手動で可能です。

## 手動ビルド

```sh
./gradlew build 
```

build/libs の中に Exsamplenなんとか.jar があるのでそれをマイクラのMODに適用。
